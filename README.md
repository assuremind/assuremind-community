## ✨ What is AssureMind?

**AssureMind Studio** is an AI-powered, codeless test automation framework that lets you write tests in plain English and automatically generates production-ready Playwright code — no scripting skills required.

### 🚀 Key Highlights

| Capability | What it does |
|---|---|
| **Zero Coding** | Write steps in plain English — AI generates Playwright code automatically |
| **MCP-Sighted Generation** | AI sees real page elements via Playwright MCP (~90-95% accuracy) |
| **3 Suite Types** | **UI** (browser) · **API** (HTTP) · **Audit** (Lighthouse) |
| **5-Level Self-Healing** | Broken selectors are auto-fixed during runs — smart retry → AI regen → multi-selector → visual → decompose |
| **12 AI Providers** | Anthropic · OpenAI · Gemini · Groq · DeepSeek · Ollama (local/free) · and more |
| **Test Recorder** | Record by clicking in a real browser — zero AI cost, zero guesswork |
| **RAG Memory** | AI learns from every run — smarter generation over time, at zero extra cost |
| **Visual Regression** | Pixel-diff screenshot comparison with baseline approval in Studio |
| **Faker Data** | 100+ realistic data generators — no more stale or duplicate test data |
| **CI-Ready** | `npx assuremind run --all --ci` — works with GitHub Actions, GitLab CI, Jenkins |

---

## ⚡ Quick Start

```bash
npm install assuremind
npx assuremind init        # sets up folders, config & Playwright browsers
npx assuremind studio      # opens Studio at http://localhost:4400
```

> Run `npx assuremind doctor` to verify your setup anytime.
